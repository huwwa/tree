# Tree

A simple reimplementation of the `tree` command.

## Sample Output

```console
$ ./bin/ctree .
.
├── LICENSE
├── Makefile
├── bin
│   ├── ctree
│   └── jtree
├── c
│   ├── tree.c
│   └── tree.h
└── java
    └── src
        └── Tree.java

5 directories, 7 files
```

## Quick Start

```console
make tree && ./tree
```
